# 📊 Blinkit Sales Analysis Dashboard

## 🔹 Project Overview
This project presents an interactive Power BI dashboard analyzing Blinkit sales data to uncover business insights, customer behavior, and performance trends.

---

## 🔹 Objectives
- Analyze total sales and revenue trends
- Understand customer purchasing behavior
- Identify top-performing categories and products
- Evaluate the impact of discounts and promotions

---

## 🔹 Key KPIs
- Total Sales
- Total Orders
- Average Order Value
- Discount Percentage
- Customer Purchase Frequency

---

## 🔹 Insights
- Sales show variation across different seasons and locations
- Discounts significantly impact purchase behavior
- Certain product categories contribute higher revenue
- Customer repeat purchases indicate loyalty trends

---

## 🔹 Tools & Technologies
- Power BI (Dashboard & Visualization)
- Data Cleaning
- Data Analysis
